Map & Reduce

Description

In 2004 Google engineers tried lots of methods to parallelize their processes.
They implemented Map and Reduce. Which drastically increased the efficiency for finding documents
and processing the whole internet. The algorithm worked by using a list of Map functions supplied to the algorithm.
The algorithm would then create key value lists that would be fed into the reduce functions.
